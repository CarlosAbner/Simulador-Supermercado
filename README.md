# SIMULADOR SUPERMERCADO

Esse projeto pessoal tem como objetivo aprimorar meus conhecimentos prévios em java.

## SOBRE O PROJETO

Tem como funcionalidade uma interação de um operador de caixa de um supermercado e um cliente ao passar suas compras no caixa.

O programa se inicia perguntando o nome do `Atendente` e o nome do `Cliente`.

O caixa pergunta o `Nome` e o `Valor` de 3 produtos. Ao Final o operador de caixa diz o total da compra.

Se o Valor total for maior que R$99,99 aplica-se um desconto de 10$ ao valor da compra.

Ao final o caixa pergunta o valor em dinheiro que o Cliente tem a entrega-lo.

O programa finaliza mostrando um `"Extrato"` de toda a operação.
